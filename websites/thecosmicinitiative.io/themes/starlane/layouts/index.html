THE COSMIC INITIATIVE

A WEBASSEMBLY ORCHESTRATION FRAMEWORK

WHAT IS IT?

A free and open source software framework written in Rust that facilitates deployment & execution of WebAssembly (Wasm) components at every level of the enterprise (Mobile, Browser, Desktop, Server, Edge, IoT.) Cosmic enabled Wasm components can discover other Wasm components & resources & interoperate via messages & remote procedure calls (RPC)

WEBASSEMBLY (Wasm)

A binary instruction format for a virtual machine.

Wasm is Fast, Safe, Portable & Ubiquitous. It has a far smaller footprint than containers (think kilobytes instead of gigabytes) and can execute in more environments than containers including the Browser, Mobile and IoT.

The Cosmic Initiative is just a set of Rust libraries to make WebAssembly shine by allowing developers to compose a software program or add on to an existing software program a framework to orhcestrate multiple Wasm components.

COSMIC FEATURES

  • ARTIFACTS

    Artifact system for bundling and deploying resource files, assets and Wasm binaries

  • MESSAGING

    A simple messaging protocol for discovering and communicating via RPC between various Wasm components.

    The Cosmic Initiative even provides a working registry and routing system for getting messages where they need to go in many environments

  • CONFIGURATION

    A configuration DSL for creating binding rules between the Wasm and its greater 'cosmic' environment.

    The binding contract ensures that the environment has everything the Wasm requires and provides an RPC interface for the Wasm

  • SECURITY

    A security model which governs between Wasm components.

    This is perhaps one of the most exciting aspects of The Cosmic Initiative: built on top of the revolutionary security of Wasm and buttressed by cosmic's chosen host engine Wasmer it should be possible for the first time to execute completely untrusted code securely when the right policy is configured with reasonable confidence--a bold claim
